backend/src/service/tenis-service.ts - servico de logica de negocio para tenis

backend/uploads/ - diretorio para armazenar arquivos enviados

frontend/ - aplicacao web do projeto

Este projeto e uma aplicacao de e-commerce para compra de tenis, desenvolvida em
Node.js com TypeScript no backend. Inclui funcionalidades de autenticacao de
usuarios, gerenciamento de categorias de produtos, pedidos e tenis, com suporte a
upload de arquivos e operacoes de banco de dados.
